copy_val/3

copy_val(Property::atom(), DestPropList::proplists:proplist(), SrcPropList::proplists:proplist()) -> proplists:proplist()

Copies the value of a property from one proplist to another. If the property is not found in the original proplist the original destination proplist is returned. This will remove all previous values associated with the property.

merge_vals/2

merge_vals(PropList1::proplists:proplist(), PropList2::proplists:proplist()) -> proplists:proplist()

Merges two proplists. The resulting proplist will preserve all key/values from proplist one. Additionally values for missing keys will be added from the second proplist.

no_badmatch/1

no_badmatch(Function::fun(() -> term())) -> term()

Evaluates the given function catching an occurence of badmatch errors. In case of badmatch the offending term is returned, in case of normal function completion its result is returned.

normalize/2

normalize(Length::non_neg_integer(), String::string() | binary()) -> binary()

read/3

read(ReadFun::fun((Offset::non_neg_integer(), Count::non_neg_integer()) -> {ReadCount::non_neg_integer(), Data::binary()}), Size::non_neg_integer(), BlockSize::non_neg_integer()) -> binary()

Generic read function that reads up to Size bytes in steps of BlockSize bytes using a provided reader function. A read operation is always expected to succeed, any exception thrown by a read will fall through to the original caller.
